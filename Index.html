How to Use:
1. Select Root
2. Tuning is fixed (Standard EADGBE)
3. Only Major scales are shown
4. Notes highlight as:
   • R = Root
   • M3/m3 = Major/Minor 3rd
   • P5 = Perfect 5th
   • M7/m7 = Major/Minor 7th
5. Click “Reset Fretboard” to start fresh
